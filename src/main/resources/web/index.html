CurrencyTrade


CurrencyTrade is a demo application

The application is intended to demonstrate the use of various technologies, specifically Scala, spray.io with websockets and mongodb, in order to expose web services and provide real-time data updates to clients

To learn more about CurrencyTrade, check out the project over at GitHub

If you don't care about the code and just want to see what the app can do, it is recommended that you open CurrencyTrade in an additional browser. Once you have two browsers open, you can use the Random trade or Bulk trades button in one browser to generate trades and then in the second browser you will be able to see the dynamic updates in the Graphs, Lists and Web Service sections

Features

Web Service

A sample trading web service is provided using spray.io, making it fast, lightweight, fully asynchronous and capable of handling thousands of concurrent connections.

Graphs

Rendered in the browser using the Flot javascript library, these graphs are updated in real-time using the latest trade data.

Lists

Updated in real-time using the latest trade data, these lists are simple html tables displaying various types of application information.

Map

Utilizing the Google Maps API, country data is displayed for each location from which a trade has been made.
